# steam-games-interactive-dashboard 📊

End-to-end data analysis project utilizing **Python (Pandas/NumPy)** for advanced data cleaning and **Tableau** for interactive BI dashboarding. Features a synchronized filtering system to identify high-value 'Hidden Gems' in the Steam marketplace.

## 🛠️ Key Features
* **Data Preprocessing:** Handled missing values and normalized rating scales using Python.
* **Synchronized Filtering:** Implemented cross-sheet filters to align Scatter Plots and Data Tables.
* **Global Context:** Designed a static distribution histogram to provide market benchmarks while filtering specific subsets.

## 📁 Repository Structure
* `notebooks/`: Jupyter Notebook with the ETL process.
* `data/`: Cleaned dataset in CSV format.
